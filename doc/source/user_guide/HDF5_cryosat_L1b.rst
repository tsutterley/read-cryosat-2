===================
HDF5_cryosat_L1b.py
===================

- Writes Level-1B CryoSat waveform data into HDF5 files
- Supported CryoSat Modes: LRM, SAR, SARin, FDM, SID, GDR
- Units for each variable should match the original SDS files
